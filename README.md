# PRC-SR

Pfingston Reception Status Board.

This is an unclassified operational throughput dashboard for tracking reception center status counts, dormitory processing status, and bus arrival totals.

## Data Rules

- No PII
- No names
- No orders data
- No trainee records
- Status counts only

## Current Build

Static HTML/CSS/JavaScript prototype.
